confity.js
=======

A simple and lightweight library to manager custom configuration for your javascript module in your web application.

## usage

#### set module configuration
```
  confiny.setModule("confiny", {
		name:"confiny",
		complex: {
			id: 10,
			value: 10,
			sub: {
				value: "Hello World !"
			}
		}
	});
```
